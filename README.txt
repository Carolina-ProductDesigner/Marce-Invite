INVITACIÓN DE BODA — MARCELA & CARLOS
=====================================

Sitio web estático, mobile-first y 100% responsivo (mobile / tablet / desktop).
Cada pantalla es una página independiente.

CÓMO VERLO
----------
Abre "index.html" en cualquier navegador. Para subirlo a un hosting
(Netlify, Vercel, GitHub Pages, etc.) sube TODA la carpeta tal cual.

PÁGINAS
-------
index.html ................ Landing (Save the date)
nos-casamos.html .......... Nos casamos
la-boda.html .............. La boda (misa, cóctail/recepción, dresscode)
hospedaje.html ............ Hospedaje
mesa-de-regalos.html ...... Mesa de regalos
confirma-asistencia.html .. Confirma tu asistencia (formulario)
gracias.html .............. Gracias por confirmar

CARPETAS
--------
/css   estilos compartidos (colores, tipografías, responsivo, animación)
/js    menú móvil y formulario
/fonts tipografías auto-alojadas (Luxurious Script + Judson)
/assets imágenes (foto novios, fondos, logo, acuarelas, hojas, etc.)

DÓNDE EDITAR
------------
- Colores y tipografías:  css/styles.css  (variables :root al inicio)
- Direcciones, horarios, códigos de hotel, links de mesa de regalos:
  edita el texto/href="#" directamente en cada archivo .html
- El formulario por ahora solo redirige a gracias.html (no guarda datos).
